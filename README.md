# ROT13-convertor
a simple web page that encrypt your text into ROT13 and can decrypt it

# Pre-requisites
python version: 3
flask

# Installing
Keep rot13.html in a folder named as "templates". 
templates and rot13.py should be in the same directory.
run rot13.py and enjoy!!
